# 關於我們頁章程區塊設計

**日期：** 2026-03-26

## 目標

在關於我們頁新增一個「協會章程」區塊，放在「我們的願景與目標」與「理監事成員」之間，提供使用者明確的章程入口。

## 版面方向

採用敘事導向的文件預覽卡：

- 延續 about 頁目前的閱讀節奏，而不是插入過度功能化的 CTA 區塊
- 以白底卡片搭配極淡暖色漸層與奶油色文件預覽，維持站點既有品牌色
- 以暖橘色按鈕作為主要操作，連到 Google Docs 章程文件並以新分頁開啟

## 內容結構

- 區塊標題：`協會章程`
- 簡短說明：說明章程涵蓋宗旨、會員制度與組織運作，作為認識協會的重要入口
- 文件感預覽面板：不嵌入真實文件內容，只用視覺暗示表現正式文件
- 主要按鈕：前往閱讀完整章程

## 樣式原則

- 主要背景維持淺色系，避免破壞關於我們頁面的整體寧靜感
- 使用 `brand`、`primary`、`secondary` 等現有色票，不新增新的品牌色系
- 卡片角度、陰影、邊框應與現有站點元件相容
- 手機版需維持單欄閱讀，按鈕明顯可點擊
